# Семантические теги

Блоки, теги, которые имеют определенные значения.  
- Теги, имеющие контекстное содержание для поисковиков.  

1. header  
2. main - главная часть
3. footer - подвал, нижняя часть  
4. nav - менюшка
5. section - разделение контента
6. article - cтатьи
